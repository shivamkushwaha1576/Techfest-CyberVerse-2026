# 🚀 Techfest CyberVerse 2026

A futuristic 3D interactive website built for the Techfest IIT Bombay CA Program.

## 🌟 Features

- Interactive 3D Globe using Three.js
- Particle Galaxy Background
- Glassmorphism Navbar
- GSAP Scroll Animations
- Custom Cursor Effect
- Typing Text Animation
- Theme Toggle (Dark/Light)
- Background Music Toggle
- Responsive Design
- Modern Cyberpunk UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- GSAP
- Three.js

## 📸 Preview

Experience a futuristic digital universe featuring immersive animations, interactive elements, and modern UI design.
